========================================
  BCHP-THERMA 서버 실행 방법
========================================

** 2개의 명령창에서 각각 실행하세요 **

1단계: RUN-FLASK.bat 더블클릭
   → Flask 서버 시작 (포트 5000)
   
2단계: RUN-NEXTJS.bat 더블클릭
   → Next.js 서버 시작 (포트 3000)
   
3단계: 브라우저 접속
   → http://localhost:3000

========================================

문제 해결:

- "pnpm을 찾을 수 없습니다" 오류
  → 명령 프롬프트에서: npm install -g pnpm
  
- "python을 찾을 수 없습니다" 오류
  → Python 3.11 이상 설치 필요

- 포트 사용 중 오류
  → 작업 관리자에서 python.exe, node.exe 종료
