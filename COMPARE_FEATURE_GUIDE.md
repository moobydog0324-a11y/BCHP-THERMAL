# 📊 비교 분석 기능 가이드

## 🎯 새로 추가된 기능

동일한 GPS 위치에서 촬영된 열화상 이미지를 **날짜별로 비교**할 수 있습니다!

---

## 🚀 사용 방법

### 1. 비교 페이지 접속
```
http://localhost:3000/compare
```

### 2. 구역 선택
- 좌측 사이드바에서 분석할 구역 선택 (예: C-1)

### 3. GPS 위치 선택
- GPS 위치 목록에서 비교할 위치 선택
- 각 위치마다 촬영된 이미지 개수 표시

### 4. 비교 모드 선택

#### 📅 **시계열 보기** (기존 모드)
- 같은 위치의 모든 이미지를 시간 순서대로 그리드 형태로 표시
- 전체 온도 범위와 시계열 변화를 한눈에 확인

#### ⚖️ **날짜 비교** (신규 모드)
- 특정 2개 날짜를 선택하여 좌우 대비
- 온도 변화 추이 자동 계산 및 시각화

---

## 📆 날짜 비교 모드 상세

### 1단계: 날짜 선택
좌측 사이드바의 **"📆 날짜 선택"** 섹션에서:
- 비교할 첫 번째 날짜 클릭 (예: 2025-03-01)
- 비교할 두 번째 날짜 클릭 (예: 2025-04-01)
- 최대 2개까지 선택 가능
- 3번째 클릭 시 가장 오래된 선택이 자동으로 해제됨

**준비 완료 표시:**
```
✅ 비교 준비 완료
2025-03-01 ↔ 2025-04-01
```

### 2단계: 좌우 대비 분석

#### 좌측: 시작 날짜 (녹색 테마)
- 📅 시작: 2025-03-01
- 해당 날짜에 촬영된 모든 이미지 표시
- 각 이미지의 최고 온도 하이라이트

#### 우측: 종료 날짜 (주황색 테마)
- 📅 종료: 2025-04-01
- 해당 날짜에 촬영된 모든 이미지 표시
- 각 이미지의 최고 온도 하이라이트

### 3단계: 온도 변화 요약

하단에 **"📈 온도 변화 분석"** 카드 자동 표시:

#### 온도 상승 케이스 (빨간색 경고)
```
📈 온도 변화 분석

시작 (2025-03-01)    종료 (2025-04-01)    변화량
   15.5°C               22.3°C           ↑ 6.8°C

⚠️ 온도가 상승했습니다. 추가 점검이 필요할 수 있습니다.
```

#### 온도 안정 케이스 (파란색)
```
📈 온도 변화 분석

시작 (2025-03-01)    종료 (2025-04-01)    변화량
   22.5°C               18.3°C           ↓ 4.2°C

✅ 온도가 안정적입니다.
```

## 🎨 UI 특징

### 색상 코딩
- **녹색 테마**: 시작 날짜 (과거)
- **주황색 테마**: 종료 날짜 (현재)
- **빨간색 경고**: 온도 상승
- **파란색 안정**: 온도 하강

### 실시간 토글
- **📷 실화상 보기**: 매칭되는 일반 사진 표시
- **🌡️ 열화상 보기**: 열화상 이미지로 복귀
- 클릭 한 번으로 즉시 전환

### 메타데이터 표시
각 이미지 카드에서 확인 가능:
- 촬영 시각 (년-월-일 시:분)
- 카메라 모델
- 최고 온도 (하이라이트)
- GPS 정보
- 주변 온도
- 날씨 상태

```
목적: 여름철 vs 겨울철 배관 온도 비교
선택: 2025-07-15 vs 2025-01-15

```

### 시나리오 2: 정기 점검 비교
```
목적: 월별 정기 점검 결과 비교
선택: 2025-03-01 vs 2025-04-01
결과: 이상 온도 상승 조기 발견
```

### 시나리오 3: 유지보수 전후 비교
```
목적: 수리 작업 효과 검증
선택: 수리 전 날짜 vs 수리 후 날짜
결과: 온도 정상화 여부 확인
```

## 📊 온도 변화 계산 방식

### 평균 최고 온도
각 날짜의 모든 이미지에서:
1. `actual_temp_stats.max_temp` 추출
2. 평균값 계산
3. 소수점 1자리까지 표시

### 변화량
```
변화량 = 종료 날짜 평균 - 시작 날짜 평균
```

- **양수 (+)**: 온도 상승 ↑ (빨간색 경고)
- **음수 (-)**: 온도 하강 ↓ (파란색 안정)

## 🔧 기술적 세부사항

### 날짜 그룹핑
```typescript
// 날짜 형식: YYYY-MM-DD
const formatDate = (timestamp: string) => {
  return timestamp.split('T')[0]
}

// 선택된 날짜의 이미지 필터링
const getImagesByDate = (date: string) => {
  return selectedGroup.images.filter(
    img => formatDate(img.capture_timestamp) === date
  )
}
```

### 온도 계산
```typescript
const getAvgMaxTemp = (images: ThermalImage[]) => {
  const temps = images
    .map(img => img.temperature.actual_temp_stats?.max_temp)
    .filter((t): t is number => t !== undefined && t !== null)
  return temps.length > 0 
    ? temps.reduce((a, b) => a + b, 0) / temps.length 
    : null
}
```

## ⚠️ 주의사항

### 메타데이터 필수
- 온도 비교를 위해서는 `thermal_data_json.actual_temp_stats`가 필요합니다
- 메타데이터가 없는 이미지는 `/data` 페이지에서 재추출하세요:
  ```
  http://localhost:3000/data
  → 🔥 온도 데이터 재추출 버튼 클릭
  ```

### GPS 정보
- GPS가 없는 이미지는 별도 그룹으로 분류됩니다
- "📍 GPS 정보 없음" 그룹도 날짜 비교 가능

### 날짜 선택
- 같은 GPS 위치 내에서만 날짜 선택 가능
- 이미지가 없는 날짜는 목록에 표시되지 않음

## 🐛 문제 해결

### "비교할 날짜를 선택해주세요" 표시
**원인**: 2개 날짜를 선택하지 않음  
**해결**: 좌측 사이드바에서 2개 날짜 선택

### "온도 변화 분석" 카드가 없음
**원인**: `actual_temp_stats` 메타데이터 누락  
**해결**: `/data` 페이지에서 온도 데이터 재추출

### 날짜 목록이 비어있음
**원인**: 해당 GPS 위치에 이미지 없음  
**해결**: 다른 GPS 위치 선택 또는 이미지 업로드
