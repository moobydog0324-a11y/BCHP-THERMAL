# 🎯 반월 열병합 열배관 관리시스템 - 간단 사용 가이드

## ✨ 3단계로 끝!

```
1. 📤 이미지 업로드 → 2. 📊 비교분석 → 3. ✅ 완료!
```

---

## 📤 1단계: 이미지 업로드

### 접속
```
http://localhost:3000/upload
```

### 진행 순서
1. **"열화상 이미지"** 클릭
2. **구역 선택** (A-1 ~ G-2 중 선택)
   - 자동으로 점검 기록 생성됨
3. **이미지 파일 선택**
   - 여러 개 한 번에 선택 가능
   - JPG, PNG, TIFF 지원
4. **업로드 클릭**
   - 자동으로 GPS, 온도, 촬영 시간 추출
   - 메타데이터 자동 저장

### ✅ 업로드 완료되면?
- 즉시 **"시간대별 비교하기"** 또는 **"GPS 기반 비교하기"** 버튼 표시
- 클릭 한 번으로 바로 분석 시작!

---

## 📊 2단계: 비교분석

### 접속
```
http://localhost:3000/compare
```

### 진행 순서
1. **구역 선택** (좌측 패널)
   - A-1 ~ G-2 중 선택
2. **날짜 선택** (최대 2개)
   - 비교하고 싶은 촬영 날짜 클릭
3. **자동으로 나란히 비교 표시**
   - 같은 GPS 위치의 이미지가 자동으로 그룹핑
   - 온도 정보 자동 표시

### 💡 자동 기능
- ✅ GPS 좌표 자동 매칭
- ✅ 시간대별 자동 그룹핑
- ✅ 온도 데이터 자동 표시
- ✅ 같은 위치 이미지 강조

## 📋 시스템 요구사항

### 필수
1. **Next.js 서버** (포트 3000)
2. **Flask 서버** (포트 5000) - 업로드 시에만 필요
3. **PostgreSQL** 데이터베이스
4. **ExifTool** - 메타데이터 추출용

### 서버 시작
```bash
# Terminal 1: Next.js
cd C:\Users\GSENR\Desktop\BCHP-THERMA
pnpm run dev

# Terminal 2: Flask (업로드 시에만 필요)
cd python-backend
python app.py
```

## ❓ 자주 묻는 질문

### Q: 이미지가 비교에 안 나와요
A: 
1. 구역이 제대로 선택되었는지 확인
2. 메타데이터가 추출되었는지 확인
3. 디버그 페이지(🐛)에서 데이터 확인

### Q: GPS 정보가 없어요
A:
- GPS 기능이 있는 열화상 카메라 사용 필요 (DJI XT2 등)
- GPS 정보 없어도 시간대별 비교는 가능

### Q: 메타데이터가 추출 안 돼요
A:
1. Flask 서버가 실행 중인지 확인
2. ExifTool이 설치되었는지 확인
3. `http://localhost:5000` 접속해서 서버 확인
